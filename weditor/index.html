À propos de Weditor

Weditor est-il le seul éditeur de texte en ligne le plus complet ?

Weditor est connu du fait de son accessibilité et qu'il est gratuit. Comme tous les produits Webine, tous ont un but précis : proposer des services pour faciliter la vie, le développement web ; c'est l'un des premiers engagements de notre société.

Comment fonctionne l'enregistrement des documents ?

Comme il n'y a pas de compte sur Weditor, pour enregistrer votre document nous utilisons les cookies plus précisément le stockage local. Votre document sera enregistré sur votre navigateur et seul vous y avez accès (uniquement sur votre navigateur actuel). Sachez que nous n'avons pas accès à vos documents.

Développement web

La plupart des produits Webine sont pour les développeurs de sites web. Sur weditor vous avez la possibilité d'obtenir de code HTML de votre document.

Documents

Tous vos documents enregistrés sont accessibles en cliquant sur "Documents". Cliquez sur "Supprimer" pour supprimer un document.

Verrouillage de l'édition

Vous avez la possibilité de verrouiller l'édition des documents avec un mot de passe. La première fois, un mot de passe temporaire vous sera demandé. Lors du déverrouillage, le mot de passe serat réinitialisé. En cas d'oubli du mot de passe il est facile de le récupérer. en nous contactant ici.

Besoin d'aide ? Nous sommes la pour vous aider.
