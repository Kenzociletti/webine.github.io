Wodal, la modal CSS la plus simple pour votre site web.

Wodal est une modal CSS pour votre site web. Tout est déjà fait et vous n'avez pas besoin de coder. Gagnez du temps, ajoutez plusieurs modal à la fois.

Comment ajouter Wodal à votre site web ?

Pour ajouter Wodal à votre site web, collez le code ci-dessous entre les balises head et /head de votre code source.

<head>
<link href="https://www.webine.fr/libs/webinecss.css" rel="stylesheet">
<script src="https://www.webine.fr/wodal/params.js"></script>
</head>

Puis, ajoutez le code HTML ci-dessous (peu importe son emplacement).

<div id="modal1" class="wodal">
  <div class="wodal-content">
    <div class="wodal-header">
      <p class="wodal-title">Titre de la modal</p>
    </div>
    <div class="wodal-body">
      <p>Contenu de la modal</p>
    </div>
    <div class="wodal-footer">
      <button id="modalbtnc1" class="btn btn-t3">Fermer</button>
    </div>
  </div>
</div>

Comment s'ouvre ma modal ?

La modal s'ouvre grâce à une fonction appelée openmodal('ID DE LA MODAL', 'ID DU BOUTON "FERMER" DE LA MODAL'). Dans cet exemple, nous utilisons un bouton pour l'ouvrir.

Exemple avec 1 modal :

<!DOCTYPE html>
<html>
<head>
<link href="https://www.webine.fr/libs/webinecss.css" rel="stylesheet">
<script src="https://www.webine.fr/wodal/params.js"></script>
</head>
<body>
 
<button class="btn btn-t1" onclick="openmodal('modal1', 'modalbtnc1');">Ouvrir la modal</button>
 
<div id="modal1" class="wodal">
  <div class="wodal-content">
    <div class="wodal-header">
      <p class="wodal-title">Titre de la modal</p>
    </div>
    <div class="wodal-body">
      <p>Contenu de la modal</p>
    </div>
    <div class="wodal-footer">
      <button id="modalbtnc1" class="btn btn-t3">Fermer</button>
    </div>
  </div>
</div>
 
</body>
</html>
