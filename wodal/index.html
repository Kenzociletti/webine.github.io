Wodal, la modal CSS la plus simple pour votre site web.

Wodal est une modal CSS et Javascript pour votre site web. Tout est déjà fait et vous n'avez pas besoin de coder ; gagnez du temps ; Ajoutez plusieurs modal à la fois.

Comment ajouter Wodal à votre site web ?

Pour ajouter Wodal à votre site web, collez le code ci-dessous entre les balises head et /head de votre code source.

<head>
<link href="https://www.webine.ga/wodal/style.css" rel="stylesheet">
<script src="https://www.webine.ga/wodal/params.js"></script>
</head>

Puis, ajoutez le code HTML ci-dessous (peu importe son emplacement).

<div id="modal1" class="cssmodal">
<div class="cssmodalcontent">
<p class="cssmodaltitle">Titre de la modal</p>
Contenu de la modal
<br><br><button id="modalbtnc1" class="cssmodalbtn cssmodalbtnclose">Fermer</button>
</div>
</div>

Comment s'ouvre ma modal ?

La modal s'ouvre grâce à une fonction appelée openmodal('ID DE LA MODAL', 'ID DU BOUTON "FERMER" DE LA MODAL'). Dans cet exemple, nous utilisons un bouton pour l'ouvrir.

Exemple avec 1 modal :

<!DOCTYPE html>
<html>
<head>
<link href="https://www.webine.ga/APIs/wodal/style.css" rel="stylesheet">
<script src="https://www.webine.ga/APIs/wodal/params.js"></script>
</head>
<body>
 
<button onclick="openmodal('modal1', 'modalbtnc1')">Ouvrir la modal</button>
 
<div id="modal1" class="cssmodal">
<div class="cssmodalcontent">
<p class="cssmodaltitle">Titre de la modal</p>
Contenu de la modal
<br><br><button id="modalbtnc1" class="cssmodalbtn cssmodalbtnclose">Fermer</button>
</div>
</div>
 
</body>
</html>
